#Todo List Full Stack Application

##Technologies Used
Backend:
- Java 17
- Spring Boot
- MySQL
- Maven

Frontend:
- React JS
- Axios
- HTML, CSS

---

##Backend Setup

1. Go to backend folder
2. Update application.properties with your MySQL username & password
3. Run:

mvn spring-boot:run

Backend runs on:
http://localhost:8080

---

##Frontend Setup

1. Go to frontend folder
2. Run:

npm install
npm start

Frontend runs on:
http://localhost:3000
---

##REST API Endpoints

GET     /tasks
POST    /tasks
PUT     /tasks/{id}
DELETE  /tasks/{id}

---

##Features

- Add Task
- Update Task
- Delete Task
- Mark as Completed
